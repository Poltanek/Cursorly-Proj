## About The Project
A full‑stack web application for uploading, validating, storing, and downloading custom cursor packs with SHA‑256 integrity protection.




### Built With
MERN Stack:
* MongoDB and utilising MongoDB Atlas Compass
* Express
* React
* Nodejs



<!-- GETTING STARTED -->
## Getting Started

  ```sh
  cd server
  npm install
  nodemon server.js
  ```

  ```sh
  cd client
  npm install
  npm run dev
  ```

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  npm install -g nodemon
  ```
