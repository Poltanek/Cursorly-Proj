# Cursorly - Capestone Project

Cursorly is a full‑stack web application designed for uploading, validating, storing, and downloading cursor packs securely. The system includes a React client for the user interface and a Node.js/Express server responsible for secure file handling, ZIP validation, hashing, and database operations. MongoDB Atlas is used as the primary database.



#  Tech Stack
Frontend: React 19, Vite, React Router
Backend: Node.js, Express, Multer, Adm‑Zip
Database: MongoDB Atlas, Mongoose
Security: SHA‑256 hashing, UUID filenames, MIME validation
Deployment: Azure App Service + Azure Static Web Apps
